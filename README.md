# Nykaa SEPM Portfolio Website

A responsive and interactive **Software Engineering Process & Management (SEPM)** portfolio website inspired by the Nykaa user interface design.  
This project showcases different phases of the **Software Development Life Cycle (SDLC)** along with UML diagrams, requirement analysis, project scheduling, and testing documentation.

---

## Features

- Responsive modern UI design
- Nykaa-inspired theme and styling
- Smooth scrolling navigation
- Scroll progress indicator
- Mobile responsive navigation menu
- Reveal-on-scroll animations
- Interactive lightbox image viewer
- UML diagram showcase
- PDF document integration
- Back-to-top button
- Academic project presentation structure
